# Dialogue Runner

Orchestration engine for running Yarn Spinner dialogues, providing runtime coordination, localization hooks, and command dispatching in TypeScript.

## Contributing
- Update [`CHANGELOG.md`](CHANGELOG.md) with every pull request. Include the release or batch date, a short summary, and categorize entries by impact (High/Medium/Low) with links to the relevant files or modules.
- Keep new functionality consistent with the existing runtime, line provider, and command dispatching patterns.
